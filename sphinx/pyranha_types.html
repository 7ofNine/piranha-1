pyranha.types¶

Types module.

pyranha.types.divisor = Type generator for the generic C++ type 'piranha::divisor'¶

Generic type generator for divisors.

pyranha.types.divisor_series = Type generator for the generic C++ type 'piranha::divisor_series'¶

Generic type generator for divisor series.

pyranha.types.double = Type generator for the C++ type 'double'¶

This generator represents the standard C++ type double.

pyranha.types.float = Type generator for the C++ type 'float'¶

This generator represents the standard C++ type float.

pyranha.types.integer = Type generator for the C++ type 'piranha::mp_integer<0>'¶

This generator represents the arbitrary-precision integer type provided by the piranha C++ library.

pyranha.types.k_monomial = Type generator for the C++ type 'piranha::kronecker_monomial<long>'¶

This generator represents the Kronecker monomial type (that is, a monomial “compressed” into a single signed integral value).

pyranha.types.long_double = Type generator for the C++ type 'long double'¶

This generator represents the standard C++ type long double.

pyranha.types.monomial = Type generator for the generic C++ type 'piranha::monomial'¶

Generic type generator for monomials.

pyranha.types.poisson_series = Type generator for the generic C++ type 'piranha::poisson_series'¶

Generic type generator for Poisson series.

pyranha.types.polynomial = Type generator for the generic C++ type 'piranha::polynomial'¶

Generic type generator for polynomials.

pyranha.types.rational = Type generator for the C++ type 'piranha::mp_rational<0>'¶

This generator represents the arbitrary-precision rational type provided by the piranha C++ library.

pyranha.types.rational_function = Type generator for the generic C++ type 'piranha::rational_function'¶

Generic type generator for rational functions.

pyranha.types.real = Type generator for the C++ type 'piranha::real'¶

This generator represents the multiprecision floating-point type provided by the piranha C++ library.

pyranha.types.short = Type generator for the C++ type 'short'¶

This generator represents the standard C++ type short (a signed integer type whose width is usually 16 bits).

pyranha.types.signed_char = Type generator for the C++ type 'signed char'¶

This generator represents the standard C++ type signed char (a signed integer type whose width is usually 8 bits).

Detailed Description

Positions class.

This is a small utility class that can be used to determine the positions, in a piranha::symbol_set a, of the symbols in the piranha::symbol_set b. The positions are stored internally in an std::vector which is guaranteed to be sorted in ascending order and which can be accessed via the iterators provided by this class. If a symbol which is present in b is not present in a, then it will be ignored.

For instance, if the set a contains the symbols [B,C,D,E] and b contains [A,B,D,F], then an instance of this class constructed passing a and b as parameters will contain the vector [0,2].

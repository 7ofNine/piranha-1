Detailed Description

template<typename T, typename U>
class piranha::has_ipow_subs< T, U >

Type trait to detect the presence of the piranha::math::ipow_subs function.

The type trait will be true if piranha::math::ipow_subs can be successfully called on instances of type T, with an instance of type U as substitution argument.

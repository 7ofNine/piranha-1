template<typename T>
struct piranha::enable_noexcept_checks< T, detail::term_enc_enabler< T > >

Specialisation of piranha::enable_noexcept_checks for piranha::term.

This specialisation is activated when T is an instance of piranha::term. The value of the type trait is set to true if both the coefficient and key types satisfy piranha::enable_noexcept_checks. Otherwise, the value of the type trait is set to false.

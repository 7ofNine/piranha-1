Detailed Description

template<typename Cf, typename Key>
class piranha::key_is_multipliable< Cf, Key >

Type trait for multipliable key.

A multipliable key satisfies the following requirements:

  • it has a static member of type std::size_t called multiply_arity with a value greater than zero,
  • it has a static function called multiply() accepting the following arguments:
    • a reference to an std::array of piranha::term of Cf and Key of size multiply_arity,
    • two const references to piranha::term of Cf and Key,
    • a const reference to piranha::symbol_set.

The decay types of Cf and Key are considered by this type trait. Cf and Key must satisfy piranha::is_cf and piranha::is_key, otherwise a compile-time error will be generated.

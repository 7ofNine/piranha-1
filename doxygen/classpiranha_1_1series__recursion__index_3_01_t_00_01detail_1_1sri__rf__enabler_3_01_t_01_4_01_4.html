Detailed Description

template<typename T>
class piranha::series_recursion_index< T, detail::sri_rf_enabler< T > >

Specialisation of piranha::series_recursion_index for piranha::rational_function.

This specialisation is enabled when T is an instance of piranha::rational_function. Although piranha::rational_function is not a piranha::series, it is assigned a recursion index of 1 in order to provide useful type coercion in a number of situations (e.g., adding a Poisson series over rational functions to a polynomial).

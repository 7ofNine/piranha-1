Detailed Description

template<typename T, typename U = T>
class piranha::is_addable< T, U >

Addable type trait.

Will be true if objects of type T can be added to objects of type U using the binary addition operator.

This type trait will strip T and U of reference qualifiers, and it will test the operator in the form

operator+(const Td &, const Ud &)

where Td and Ud are T and U after the removal of reference qualifiers. E.g.:

is_addable<int>::value == true;
is_addable<int,std::string>::value == false;

Detailed Description

template<typename Archive, typename T>
struct piranha::boost_load_impl< Archive, T, rf_boost_load_enabler< Archive, T > >

Specialisation of piranha::boost_load() for piranha::rational_function.

Note
This specialisation is enabled only if T is an instance of piranha::rational_function whose numerator/denominator type satisfies piranha::has_boost_load.

If Archive is boost::archive::binary_iarchive, no checking is performed on the content of ar. Otherwise, it will be ensured that the deserialized rational function is in canonical form.

Exceptions
unspecifiedany exception thrown by piranha::boost_load() or by the constructor of piranha::rational_function from numerator and denominator.

Call operator.

The body of this operator is equivalent to:

ar >> x;

That is, the content of ar will loaded into x using the stream extraction operator. In order for this method to be callable, the type T must provide valid overloads of the methods/functions needed by the Boost serialization API.

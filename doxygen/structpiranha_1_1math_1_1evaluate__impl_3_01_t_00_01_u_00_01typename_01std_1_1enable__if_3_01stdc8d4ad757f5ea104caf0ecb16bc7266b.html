Note
This operator is enabled only if the algorithm described below is supported by the involved types, and it returns a type which satisfies piranha::is_returnable.

The evaluation of a rational function is constructed from the ratio of the evaluation of its numerator by the evaluation of its denominator. The return type is the type resulting from this operation.

Parameters

Detailed Description

template<typename T>
struct piranha::is_mappable< T >

Detect types that can be used as mapped values in associative containers.

This type trait is intended to detect whether T supports typical operations that can be performed on the mapped values in associative containers. Specifically, this trait will be true if all the following conditions hold:

  • T is default constructible,
  • T is copy constructible and assignable,
  • T is move constructible and assignable,
  • T is destructible.

Otherwise, the value of this trait will be false.

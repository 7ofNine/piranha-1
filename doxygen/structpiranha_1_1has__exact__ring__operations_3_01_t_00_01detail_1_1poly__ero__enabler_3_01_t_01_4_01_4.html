template<typename T>
struct piranha::has_exact_ring_operations< T, detail::poly_ero_enabler< T > >

Exact ring operations specialisation for piranha::polynomial.

This specialisation is enabled if the decay type of T is an instance of piranha::polynomial whose coefficient type satisfies piranha::has_exact_ring_operations and whose exponent type is either a C++ integral type or another type which satisfies piranha::has_exact_ring_operations.

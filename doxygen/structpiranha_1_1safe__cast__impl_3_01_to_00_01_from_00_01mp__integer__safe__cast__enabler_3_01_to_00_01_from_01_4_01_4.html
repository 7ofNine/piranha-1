Detailed Description

template<typename To, typename From>
struct piranha::safe_cast_impl< To, From, mp_integer_safe_cast_enabler< To, From > >

Specialisation of piranha::safe_cast() for conversions involving piranha::mp_integer.

Note
This specialisation is enabled in the following cases:
  • To is an instance of piranha::mp_integer and From is a C++ floating-point type from which piranha::mp_integer can be constructed,
  • To is an instance of piranha::mp_integer and From is a C++ integral type from which piranha::mp_integer can be constructed,
  • From is an instance of piranha::mp_integer and To is a C++ integral type constructible from piranha::mp_integer.

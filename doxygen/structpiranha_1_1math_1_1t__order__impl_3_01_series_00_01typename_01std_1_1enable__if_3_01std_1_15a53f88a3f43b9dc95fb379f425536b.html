Detailed Description

template<typename Series>
struct piranha::math::t_order_impl< Series, typename std::enable_if< std::is_base_of< detail::trigonometric_series_tag, Series >::value >::type >

Specialisation of the piranha::math::t_order() functor for instances of piranha::trigonometric_series.

This specialisation is activated if Series is an instance of piranha::trigonometric_series. The corrsponding method in piranha::trigonometric_series will be used for the computation, if available.

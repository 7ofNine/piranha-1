Detailed Description

template<typename T>
class piranha::key_is_differentiable< T >

Type trait to detect differentiable keys.

This type trait will be true if Key is a key type providing a const method partial() taking a const instance of piranha::symbol_set::positions and a const instance of piranha::symbol_set as input, and returning an std::pair of any type and Key. Otherwise, the type trait will be false. If Key does not satisfy piranha::is_key, a compilation error will be produced.

The decay type of Key is considered in this type trait.

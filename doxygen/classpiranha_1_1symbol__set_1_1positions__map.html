Detailed Description

template<typename T>
class piranha::symbol_set::positions_map< T >

Positions map class.

This class is similar to piranha::symbol_set::positions. In addition to storing the positions of the symbols from a map b with respect to a reference piranha::symbol_set a, it will also store the instances of T that were originally mapped to the symbols in b.

For instance, if T is int, a contains the symbols [B,C,D,E] and b is the map [(A,10),(B,20),(D,30),(F,40)], then an instance of this class constructed passing a and b as parameters will contain the vector [(0,20),(2,30)]. The internal vector is guaranteed to be sorted according to the position of the symbols.

Type requirements

T must be destructible, copy-constructible, move-constructible and move-assignable.
